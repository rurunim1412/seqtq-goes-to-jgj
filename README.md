Website dashboard interaktif untuk memandu perjalanan *road trip* dari Jakarta ke Yogyakarta. Proyek ini dirancang untuk memberikan informasi jadwal secara *real-time*, rute perjalanan, dan dokumentasi visual bagi seluruh peserta perjalanan.

## 🌟 Fitur Utama

- **📍 High-Fidelity Timeline**: Menampilkan 25 detail agenda secara kronologis menggunakan komponen *Accordion* yang rapi.
- **⚡ Live Progress Highlight**: Script otomatis yang mendeteksi waktu saat ini dan memberikan penanda visual pada agenda yang sedang berlangsung.
- **📸 Photography Gallery**: Galeri responsif dengan fitur *lightbox* untuk menampilkan dokumentasi destinasi wisata (Pantai Drini, Sri Gethuk, dll).
- **📱 Mobile First Design**: Dioptimalkan untuk penggunaan di smartphone selama di perjalanan.
- **🌙 Night-Mode Schedule**: Skema warna gelap pada bagian jadwal untuk kenyamanan mata saat diakses malam hari.

## 🛠️ Stack Teknologi

- **Frontend**: HTML5, CSS3 (Custom Styles)
- **Framework**: [Bootstrap 5.3](https://getbootstrap.com/)
- **Interactivity**: Vanilla JavaScript (ES6+)
- **Icons**: [FontAwesome](https://fontawesome.com/) / Bootstrap Icons
